# نظام إدارة المدرسة

## عن المشروع

هذا المشروع عبارة عن نظام قاعدة بيانات بسيط لإدارة أهم العمليات داخل المدرسة. الهدف من المشروع هو تنظيم البيانات المتعلقة بالطلاب، المعلمين، الصفوف، والامتحانات، وتقليل الاعتماد على الطرق التقليدية غير المنظمة.

تم تنفيذ المشروع باستخدام نظام ادارة قواعد بيانات مع التركيز على تصميم الجداول والعلاقات بشكل واضح وقابل للتطوير لاحقًا.

---

## المكونات التقنية

* نظام إدارة قواعد البيانات: MySQL
* لغة الاستعلامات: SQL
* أدوات العمل: MySQL Workbench

---

## ملفات المشروع

 schema.sql   يحتوي على أوامر إنشاء الجداول والعلاقات بين الكيانات 
 data.sql     بيانات تجريبية لاختبار الجداول              
 queries.sql  مجموعة استعلامات للتجربة والفهم                      

---

## طريقة التشغيل

### 1. إنشاء قاعدة البيانات

sql
CREATE DATABASE school_db;
USE school_db;

MySQL Workbench يمكن تنفيذ الأوامر السابقة من خلال 
CMD أو من سطر الأوامر.

---

### 2. إنشاء الجداول

بعد إنشاء قاعدة البيانات، يتم تنفيذ ملف `schema.sql` لإنشاء الجداول والعلاقات.

MySQL Workbench افتح الملف باستخدام 
 نفّذ جميع الأوامر الموجودة داخله

MySQL-CMD أو من داخل :

SOURCE [C:/path/to/schema.sql;]

---

### 3. إدخال بيانات تجريبية (اختياري)

ملف يحتوي على بيانات بسيطة لاختبار عمل الجداول والعلاقات.`data.sql`

---

### 4. تجربة الاستعلامات

 ملف  يحتوي على استعلامات جاهزة لفهم طريقة التعامل مع البيانات، `queries.sql`
